### 🚀 具体的运行命令

当前项目已经具备了可运行的基础环境，你可以按照以下两种方式之一来运行项目（暂时无需实现具体功能）：

#### 方式一：本地开发模式（推荐）

这种方式可以让你在本地终端直接查看前后端的运行日志，并且利用 Vite 的热更新机制进行开发。

**1. 启动数据库与缓存中间件**
打开终端，在项目根目录 `/Users/bytedance/Workspace/05_Code/FundTogether` 下执行：
```bash
docker-compose up -d mysql redis
```
*(这会启动运行在 3306 端口的 MySQL 和运行在 6379 端口的 Redis)*

**2. 启动后端服务**
数据库启动成功后，在终端执行以下命令启动 Spring Boot（确保本地已配置 JDK 17）：
```bash
cd backend
./mvnw spring-boot:run
```
*(服务将在 `localhost:8080` 启动)*

**3. 启动前端服务**
新开一个终端窗口，执行以下命令启动 Vue3 前端：
```bash
cd frontend
npm run dev
```
*(服务将在 `http://localhost:5173/` 启动，你可以直接在浏览器中访问该地址)*

#### 方式二：全栈容器化一键部署

如果你希望完全使用 Docker 容器来运行包括前后端在内的所有服务，可以在项目根目录直接执行：
```bash
docker-compose up -d --build
```
*(Docker 会自动拉取镜像并编译前后端代码，最终前端会暴露在 `localhost:80` 端口，后端在 `localhost:8080`)*
